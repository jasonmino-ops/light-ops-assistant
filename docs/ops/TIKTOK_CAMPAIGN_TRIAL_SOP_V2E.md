# TikTok 推广模块 · 试跑运营 SOP v2E

**版本：** v2E  
**日期：** 2026-05-31  
**适用范围：** 商户首次 TikTok 试跑 + 博主推广素材使用  
**技术基线：** v2D（推广短链 + 博主看板 + 推广素材抽屉）

---

## 一、商户试跑前准备

**预计耗时：10 分钟**

### Step 1 — 选择一个真实商户

选择一个已在系统中开通的店铺，确认：
- 商品已上架，菜单可正常打开（`/menu`）
- 店铺 Banner 已上传（影响落地页视觉效果）
- 公告文案已填写（落地页会展示）

### Step 2 — 创建博主/推广人

进入 `/campaign` → 「博主管理」→「+ 新增博主」，填写：
- **博主名称**（必填）：博主真实名或昵称
- **TikTok 账号**（选填）：填写会显示在博主看板
- **默认语言**：根据博主语言偏好选择（中文 / English / ខ្មែរ）

### Step 3 — 生成推广短链

进入「生成推广短链」区域：
- 选择刚创建的博主
- 填写视频标题备注（如：5月新品探店视频）
- 佣金规则按需填写（试跑阶段可暂不设置）
- 点击「生成短链」

记下短链码，例如 `/v/abc123`，完整链接为 `https://yourdomain.com/v/abc123`

### Step 4 — 打开推广素材

在「历史短链」区域，找到刚生成的短链 → 点击「📱 素材」按钮，抽屉弹出后：

### Step 5 — 复制 Bio 文案

在素材抽屉中，选择博主使用的语言，复制「TikTok 主页 Bio 文案」：
```
👇 点主页链接进店下单
TikTok 粉丝专享价
```
发送给博主，用于更新 TikTok 主页个人简介（Bio）。

### Step 6 — 复制置顶评论文案

复制「视频置顶评论文案」，发送给博主，在视频发布后置顶在评论区。

### Step 7 — 下载/截图二维码

在素材抽屉中长按或截图二维码（160×160 px），发送给博主用于视频结尾展示。建议博主将二维码打印或显示在手机/平板上拍摄。

### Step 8 — 确认落地页正常

用手机浏览器打开 `/v/[code]`，确认：
- 页面正常加载，显示店铺名和 Banner
- 「立即下单」和「查看菜单」按钮可点击并跳转正确页面
- 显示博主推荐信息（如已设置）
- 三语切换正常（如博主有外国粉丝需求）

## 二、TikTok 挂链方式

### 主入口：主页 Bio 链接

- TikTok 个人主页 → 编辑资料 → 网站链接
- 填入完整短链：`https://yourdomain.com/v/abc123`
- Bio 文字配合引导，例如：「👇 点主页链接下单」

**重要：** 这是唯一可以被真正点击的链接入口，是最重要的流量通道。

### 视频标题

- 仅做内容引导，如：「想知道在哪买？主页链接进店 👆」
- 不要在标题放完整 URL（超长且无法点击）

### 置顶评论

- 只做引导文字，如：「想下单？点主页第一个链接 👆 TikTok 粉丝专属」
- 不依赖评论中的链接可点击（TikTok 评论链接点击率极低且因版本而异）
- 短链 URL 可以写出来，作为补充，不是主路径

### 视频结尾二维码

- 最后 2-3 秒展示二维码（手持或贴屏幕）
- 配合口播：「扫码也可以直接下单」
- 适合不会点 Bio 链接的用户兜底

### 评论互动引导

- 用户在评论区评论「菜单」「在哪买」「链接」等关键词时，由人工统一回复私信
- 私信模板见第六节

## 三、推荐视频结构

```
┌─────────────────────────────────────────────────┐
│  0–3 秒：钩子（Hook）                            │
│  直接抛出核心吸引点：价格、奇特品、环境氛围       │
│  例："这家店的珍珠奶茶竟然只要 $2！"            │
├─────────────────────────────────────────────────┤
│  3–15 秒：展示内容                               │
│  产品特写 + 环境 + 价格标签 + 人物反应            │
│  重点展示「为什么值得去」                         │
├─────────────────────────────────────────────────┤
│  15–20 秒：行动引导                               │
│  口播："想下单或者看菜单，点我主页第一个链接 👆"  │
│  字幕辅助：「主页链接直达」                       │
├─────────────────────────────────────────────────┤
│  最后 2–3 秒：二维码特写                          │
│  手持手机/纸张，镜头聚焦二维码                   │
│  口播："也可以直接扫码"                          │
└─────────────────────────────────────────────────┘
```

**视频时长建议：** 15–30 秒（TikTok 完播率更好）  
**字幕建议：** 全程加字幕，提升完播率  
**BGM：** 选用 TikTok 当前热门音乐

## 七、试跑数据记录方法

使用 `/campaign` 页面现有数据，无需新增字段。

### 在哪里看数据

1. 进入 `/campaign`
2. 「博主汇总」区域查看每位博主的整体表现
3. 「历史短链」区域查看每条短链的明细数据

### 数据字段说明

| 字段 | 含义 | 观察重点 |
|------|------|---------|
| **浏览数** | `/v/[code]` 落地页被打开的次数 | 反映视频触达和 Bio 引流效果 |
| **点击数** | 落地页点击"立即下单"或"查看菜单"的次数 | 反映落地页转化意愿 |
| **订单数** | 通过短链归因的实际下单次数 | 核心成交指标 |
| **成交金额** | 归因订单的合计金额 | 推广商业价值 |
| **预计佣金** | 按设置规则计算的博主佣金 | 博主激励参考 |
| **结算状态** | 已结算 / 未结算 | 人工确认打款后标记 |

### 数据读取频率建议

| 阶段 | 频率 |
|------|------|
| 视频发布后 24 小时内 | 每 4-6 小时看一次浏览和点击 |
| 发布后 2-7 天 | 每天看一次，关注订单累积 |
| 第一轮结束（7 天） | 做一次综合复盘 |

## 八、简单判断规则

| 现象 | 可能原因 | 建议动作 |
|------|---------|---------|
| 浏览高、点击低 | 视频结尾引导不清楚；落地页首屏不够直观 | 优化视频 CTA 话术；检查落地页 Banner 和公告 |
| 点击高、订单低 | 落地页商品吸引力不足；价格不清楚；菜单操作复杂 | 更新公告突出特价；检查菜单商品图和定价 |
| 订单高、客单价低 | 单品价格低；用户以小额测试为主 | 推荐套餐或满减活动，引导提高客单价 |
| 博主播放量高但成交低 | 博主粉丝画像与店铺客群不匹配 | 换品类更匹配的博主；或调整推广商品 |
| 评论区互动多但点击少 | Bio 链接入口不够显眼；评论引导文案需加强 | 确认 Bio 链接已设置；更新置顶评论引导文字 |
| 浏览持续增长但无转化 | 非目标区域的流量（外地/其他国家） | 在视频中明确提及服务城市/区域 |

## 九、第一轮试跑建议

### 规模控制

```
商户数：   1 家（验证流程，不扩大变量）
博主数：   1-3 人（对比不同风格效果）
每人短链： 1 条（每人单独追踪）
每人视频： 1-3 条（同一条链接挂多个视频）
观察周期： 连续 3 天（TikTok 推荐期）
```

### 时间节点

| 天数 | 动作 |
|------|------|
| D0  | 完成商户准备、生成短链、发送素材给博主 |
| D1  | 博主发布第一条视频，确认 Bio 链接和置顶评论正确 |
| D1-D3 | 监控浏览/点击，人工回复评论私信 |
| D3  | 看第一轮数据，初步判断是否继续/调整 |
| D7  | 第一轮完整复盘，参考第八节判断规则 |

## 十、明确不做内容

本轮试跑 SOP 范围内**不涉及**以下内容：

| 项目 | 说明 |
|------|------|
| TikTok API 对接 | 播放量/粉丝/评论数需人工查看 TikTok 后台 |
| 自动私信 | 评论回复全部人工操作 |
| 评论链接可点击性保证 | TikTok 评论链接点击行为因版本和账号资质而异，不承诺 |
| 复杂分销 / 多级代理 | 当前仅支持一级博主佣金 |
| 实时数据同步 | 数据更新频率取决于用户访问落地页，不做推送告警 |
| 自动对账 / 自动打款 | 结算状态由商户人工在 /campaign 标记 |
